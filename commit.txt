- Implemented CreateSeatCommand, CreateSeatCommandHandler, and CreateSeatCommandValidator.
- Implemented GetSeatByIdQuery, GetSeatByIdQueryHandler, and GetSeatByIdQueryValidator.
- Created corresponding DTOs (CreateSeatDto, SeatDto).
- Added SeatProfile for AutoMapper.
- Implemented SeatController with CreateSeat and GetSeatById endpoints.
- Added comprehensive unit tests for all new components.
